Personaliza la IA en 4 pasos

Aitrain esta diseñado para que uses esta web conectada a un backend local corriendo en tu computadora. Usamos Docker para esto, asi que si no lo tienes deberas descargarlo. Descargar Docker 

                ##docker-compose.yml
                version: '3.8'
                services:
          
                  db:
                    image: postgres
                    environment:
                      POSTGRES_DB: aitrain
                      POSTGRES_USER: postgres
                      POSTGRES_PASSWORD: aitrainPass
                    ports:
                      - "6432:5432"
                    volumes:
                      - postgres_data:/var/lib/postgresql/data
          
                  app:
                    image: juliodaza/aitrain-backend:latest
                    ports:
                      - "3900:3900"
                    environment:
                      DATABASE_URL: 
                        "postgresql://postgres:aitrainPass@db:5432/aitrain"
                    depends_on:
                      - db
          
                volumes:
                  postgres_data:

Pasos

1

Copia el docker compose y ejecutalo: Con el archivo se bajara la ultima imagen de aitrain-backend y se crearan dos servicios: uno con la base de datos de Postgres y la otra con el backend de Node.

2

Configura el entorno: En esta web ve a configuracion y agrega la URL de backend y la key de OpenAI. Si no modificaste nada el backend corre en http://localhost:3900

3

Crea Doc Trainers: Ve a Doc Trainers y en crear docs sube los archivos que usaras de base para entrenamieno, llena el formulario y Aitrain generará los archivos .jsonl para entrenar la IA.

4

Entrena tu Modelo: en IA Train crea un nuevo entrenamiento usando los docs de entrenamiento, espera unos minutos y ya tendras un Modelo personalizado con tus datos.
